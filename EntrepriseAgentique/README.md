# Entreprise Agentique

Bienvenue dans ce dépôt de documentation sur le concept d'**Entreprise Agentique**.

L'Entreprise Agentique représente une évolution majeure des architectures d'entreprise, s'appuyant sur plusieurs piliers technologiques convergents :

- **Event-Driven Architecture (EDA)** avec **Kafka** pour une gestion réactive et découplée des flux d'information.
- **AgentOps** pour le déploiement, la gestion et la surveillance des agents logiciels autonomes.
- **Intelligence Artificielle** pour doter les processus d'une capacité de décision et d'adaptation.
- **Informatique Quantique** en tant qu'horizon technologique pour la résolution de problèmes complexes.
